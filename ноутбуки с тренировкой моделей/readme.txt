В данной папке содержатся два ноутбука с тренировкой моделей. В каждом из ноутбуков решается отдельная задача.

CycleGAN_summer_winter_transform.ipynb - рассматривается существующая задача переноса времени года (так же здесь приведены все объснения  построенной архитектуры)

CycleGAN_race_transform.ipynb - рассматривается новая задача переноса цветовой палитры с одной рассы на другую.
